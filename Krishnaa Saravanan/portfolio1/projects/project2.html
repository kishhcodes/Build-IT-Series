QUEST OVERVIEW

The Legend of Code is a full-stack application inspired by the adventure and exploration of The Legend of Zelda series. This project combines modern web technologies with classic game design principles.

ITEMS USED

React
Node.js
MongoDB

THE QUEST

KOKIRI FOREST: CONCEPT

Like Link beginning his journey in the forest, this project started with a clear vision: create an application that guides users through a narrative experience while solving real-world problems.

HYRULE FIELD: DEVELOPMENT

Development expanded across the open fields of possibilities, using React for the front-end interface, Node.js for server-side logic, and MongoDB for data persistence—creating a seamless and immersive user experience.

APPLICATION SCREENSHOT

GANON'S TOWER: OBSTACLES

The final boss of this project was implementing real-time data synchronization between users. After many attempts, I conquered this challenge using Socket.IO and careful state management.

TRIFORCE COMPLETE: RESULTS

The three pieces of the Triforce—Performance, Usability, and Innovation—came together in the completed project. Users praised the intuitive navigation and engaging interactive elements.
