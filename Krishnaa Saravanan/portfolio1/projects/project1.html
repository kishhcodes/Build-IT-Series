GAME OVERVIEW

Super Project World is a Super Mario-inspired web application that showcases my front-end development skills. It features responsive design, interactive elements, and smooth animations.

POWER-UPS USED

HTML5
CSS3
JavaScript

THE ADVENTURE

WORLD 1-1: PLANNING

The journey began with sketching layouts and wireframes, drawing inspiration from the colorful worlds of Mario. I wanted to create an experience that was both nostalgic and modern.

WORLD 1-2: DESIGN

For the visual design, I utilized a vibrant color palette reminiscent of the Mushroom Kingdom, with careful attention to pixel-perfect details and playful typography.

WORLD MAP SCREENSHOT

BOSS BATTLE: CHALLENGES

The biggest challenge was implementing responsive design that maintained the pixel art aesthetic across devices. I solved this by using CSS grid and careful media queries.

VICTORY FLAG: OUTCOME

The final project received excellent feedback for its playful interface and smooth user experience. Users particularly enjoyed the subtle animation effects and Easter eggs.
